# [Key Promoter X](https://plugins.jetbrains.com/plugin/9792-key-promoter-x)

Key Promoter X 是一款快捷键提示插件，如果你进行了**可以使用快捷键但是并没有使用快捷键**触发的操作，插件会自动在屏幕右下角提示可以使用哪些快捷键来触发同样的操作。

*如果你觉得某个提示对你造成了打扰，也可以单独禁用指定快捷键的提示。*
